# 🌐 Portfolio Website

This is my personal portfolio website showcasing my skills, projects, and interactive features.  
It is built using HTML, CSS, and JavaScript with a modern UI and animations.

---

## 🚀 Live Demo
👉 https://aaliya9051.github.io/portfolio-website/

---

## ✨ Features

- 👩‍💻 Personal portfolio homepage
- 📂 Projects showcase
- 📝 To-Do List App (with progress tracking)
- 📊 Dashboard with charts
- 🤖 AI-based Q&A section
- 🌙 Dark theme UI
- ✨ Smooth animations and effects
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (DOM Manipulation, Local Storage)  
- Chart.js (for dashboard charts)
